# Análise-de-Sentimento-com-Machine-Learning
Projeto criado através do curso gratuito DS Academy (Fundamentos de Linguagem Python - Do Básico a Aplicações de IA / DSA-Python-MiniProjeto6 ) onde o objetivo é utilizar um modelo supervisionado para identificar o sentimento de um comentário do usuário.
